# Titanic Neural Network

This repository contains a neural network built from scratch to predict passenger survival on the Titanic dataset. It demonstrates the full machine learning workflow, including data preprocessing, model design, training, evaluation, and saving the trained model for reuse.

## 📂 Project Structure
- **titanic_nn.py** – Python script implementing the neural network architecture and training logic  
- **Titanic.csv** – Dataset used for training and testing  
- **titanic_model.h5** – Saved trained model for inference  
- **.vscode/** – Development environment configuration  

## ✨ Features
- Custom neural network implementation without high-level frameworks  
- Preprocessing of Titanic dataset features for effective training  
- Model evaluation and persistence in `.h5` format  
- Modular code structure for clarity and extension
